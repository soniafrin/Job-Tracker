
## Answers to Questions

### 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
Ans:- getElementById - access to the element with ID
	- getElementsByClassName - access to the elements or element with class name.
	- querySelector / querySelectorAll - access to the elements or element 	with ID, class or tag name
